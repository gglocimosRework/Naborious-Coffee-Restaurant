# Naborious Coffee

A simple website of a fictional coffee & brunch restaurant, created using HTML and CSS.  
This project is a work-in-progress and will be updated with new features and sections.

## Features
- Responsive header and navigation
- Featured coffees & Canadian sweets sections
- Modern hover effects on items
- Mobile-friendly layout

## How to view
1. Clone the repository:

git clone https://github.com/gglocimosRework/Naborious-Coffee-Restaurant.git
2. Open Naborious-coffee.html in your browser or view the live site here:

- [GitHub Pages](https://gglocimosrework.github.io/Naborious-Coffee-Restaurant/)
